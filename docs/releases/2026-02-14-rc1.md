## Deploy Steps (Linux)
1. Copy tarball to host.
2. Verify checksum against `SHA256SUMS.txt`.
3. Extract and run from `bin/`.

Example:
```bash
sha256sum tmax-x86_64-unknown-linux-gnu.tar.gz
mkdir -p /opt/tmax
sudo tar -xzf tmax-x86_64-unknown-linux-gnu.tar.gz -C /opt/tmax
/opt/tmax/tmax-x86_64-unknown-linux-gnu/bin/tmax-local
```
